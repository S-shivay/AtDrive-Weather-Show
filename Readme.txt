# Angular Weather Application

This is an Angular-based Weather application that includes latitude and longitude for live weather updates.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/en/download/) (v12.x or higher)
- [Angular CLI](https://angular.io/cli) (v14 or higher)
- [npm](https://www.npmjs.com/get-npm) (Node Package Manager)

## Installation

### 1. Clone the repository

First, clone the repository to your local machine using Git or download it as a ZIP.


### 2. Go to the project directory by command cd project-directory

### 3. Run Command ng serve 

### 4. open browser and run http://localhost:4200
